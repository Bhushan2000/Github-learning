.net

.NET (pronounced dot net) is a framework that provides a programming guidelines that can be used to develop a wide range of applications from web to mobile to Windows-based applications.
The . NET framework can work with several programming languages such as C#, VB.NET, C++ and F#.

I heard that .NET was only for Windows or PC users. Is that true? Not anymore! Thanks to .NET Core, a set of tools consisting of the runtime, library and compiler components, you can create apps that run on Windows, Mac OS X and Linux.
.NET Core runtime, libraries, and compiler are all open source on GitHub and are taking contributions.
